# 任务依赖关系图

## Phase 0: 核心通路 (1周)
```
数据库设计 → 文件扫描器
数据库设计 → 哈希系统
数据库设计 → FTS搜索
数据库设计 → UI框架
文件扫描器 → 预览服务
哈希系统 → 相似度算法
FTS搜索 → SavedView引擎
```

## Phase 1: MVP (2-3周)
```
Phase 0 完成 → 预览服务
Phase 0 完成 → 相似度算法
Phase 0 完成 → SavedView引擎
Phase 0 完成 → 容器提取
```

## Phase 1.5: 规则引擎 (2-3周)
```
Phase 1 完成 → 规则引擎
数据库设计 → 任务队列
任务队列 → 解析器沙箱
```

## Phase 2: 专业化 (1-2个月)
```
Phase 1.5 完成 → 关系推断
Phase 1.5 完成 → 虚拟挂载
数据库设计 → 审计系统
审计系统 → 备份快照
```

## 关键路径
1. **数据库设计** - 所有功能的基础
2. **文件扫描器** - 数据采集入口
3. **哈希系统** - 去重和相似度基础
4. **UI框架** - 用户交互界面
5. **FTS搜索** - 核心检索功能

## 并行开发任务
- 数据库设计完成后，可以并行开发：
  - 文件扫描器
  - 哈希系统
  - FTS搜索
  - UI框架

- Phase 0 完成后，可以并行开发：
  - 预览服务
  - 相似度算法
  - SavedView引擎
  - 容器提取
